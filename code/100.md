# 100. 增减序列 :
题意： 给你一个数组A，每次你可以使区间 [l:r] 增加或减少1， 问最少次数使所有数字一样，并且有多少种最终可能的答案<br/>

**Ans1** : <br/>
我们可以考虑没相邻两数的差，let **B[i]=A[i]-A[i-1]**<br/>
如果我们对 [l:r]进行操作，那么 **B[l]+1** & **B[r+1]-1** <br/>
由此性质，我们可以谈心的对 **B** 的正负进行匹配 使得 **B[1:n-1]** 都是0<br/><br/>

**Ans2** : <br/>
**Math.abs(pos-neg)** 是剩余无法匹配的，我们得用第一个数字来进行额外操作，会有额外的一个 <br/>
**Math.abs(pos-neg)**+1

Time : **O(n)**<br/>
Space : **O(n)**<br/>

class Sol{
    public void solution(PrintWriter out,int A[]){
        int B[]=new int[A.length];
        long neg=0,pos=0;
        for(int i=1;i<A.length;i++){
            B[i]=A[i]-A[i-1];
            if(B[i]<0)neg-=B[i];
            else pos+=B[i];
        }

        out.println(Math.min(pos,neg)+Math.abs(pos-neg));
        out.println(Math.abs(pos-neg)+1);
    }







}
